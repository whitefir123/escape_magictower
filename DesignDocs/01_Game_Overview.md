# 逃离魔塔 (Escape the Magic Tower) - 游戏核心概述文档

## 一、 游戏基础设定与目标

### 1.1 核心体验与题材
- **游戏题材**：类魔塔的黑暗幻想风格的地下城探索游戏。
- **核心定位**：Roguelike RPG，结合了魔塔策略战斗与基于“装备词缀刷取（Diablo-like）”的长期养成。
- **视听感知**：拥有战争迷雾探索、环境光暗变化，以及明显的元素反应攻击反馈体验。

### 1.2 游戏核心目标
- **局内目标**：在程序化生成的地下城中，击败怪物、开启宝箱、与特殊NPC（商人、赌徒、铁匠）进行资源置换，尽可能深入更高的楼层。
- **局外目标**：通过死亡或通关结算获取元资源（水晶），解锁永久的天赋树；通过不断挑战获取高品质底材和专属套装，挑战极限的“噩梦层级”以验证战力。

## 二、 核心循环剖析 (Gameplay Loop)

### 2.1 局内探索循环 (In-Run Loop)
1. **地图探索**：
   - 玩家进入被“战争迷雾”覆盖的当前楼层。
   - 只能看见已探索区域，房间门未开启前无法得知内部存在精英怪还是宝箱。
   - 在走廊中移动存在触发陷阱或暗雷遭遇战的设定。
2. **回合制战斗**：
   - 当玩家在地图网格中触发怪物阻挡时，进入战斗状态。
   - 执行攻击、释放技能、触发被动天赋及进行元素异常状态计算的循环。
3. **奖励结算与强化尝试**：
   - 击杀怪物时直接获取经验（EXP）与金币，并有 **40% 概率触发【强制暂停：纯属性强化符文三选一】**；同时概率掉落带有品质与随机词缀的装备。
   - 玩家每一次积攒经验升级时，获得**全基础属性+1**，并必定弹出“机制符文系统（Roguelike天赋三选一）”，选择强大的核心机制突变。
   - 在地图中遭遇铁匠铺等设施，消耗金币对刚获得的装备进行保值强化。
4. **推进楼层**：
   - 找到通往下层的阶梯并确认进入。随着楼层加深，怪物基础面板按既定系数膨胀。

### 2.2 局外元养成循环 (Meta-Progression Loop)
1. **死亡与结算**：
   - 玩家一旦生命值归零即判定单局失败（无复活机制），进入结算界面。
   - 系统根据玩家到达的层数、击杀的精英怪/首领数量折算全局代币“水晶”。
2. **星盘天赋升级**：
   - 回到主神位（主菜单）进入天赋系统。
   - 消耗水晶点亮大树节点，提升跨局生效的属性上限（例如：初始金币增加、暴击率底座提升）。
3. **挑战天梯解锁**：
   - **噩梦层级（Ascension）解锁**：初始默认等级为 1。当玩家成功通关当前最高层级后，将解锁下一层级，最高可挑战至 25 层。
   - **难度阶梯式增长**：每一层级都会在保留前置层级难度的基础上，叠加新的负面词条（如：精英怪生成概率 +3%、商店价格提升、首领怪获得额外护盾等），持续压榨玩家的策略极限。

## 三、 世界观与通用规则

### 3.1 随机种子的绝对统治 (Deterministic RNG)
- 游戏内所有的地牢房间分布、怪物刷新位、宝箱内容、击杀掉落、游戏机抽奖结果、商店刷新的物品，乃至升级时的3个符文选项，均严格受控于“全局随机种子”。
- 在相同的种子输入下，游戏的全流程推演结果必须是100%固定复现的。

### 3.2 噩梦层级难度成长 (Ascension Nightmare Tiers)
- 属于游戏的高阶进阶系统，包含25级以上逐步递增的难度词条。
- **运作逻辑**：
  - 玩家可在菜单手动选择已解锁的最高或向下兼容的噩梦等级。
  - 等级附加的是“全局乘区补偿”与“机制突变”。
  - 例如：1~5级提高基础怪物面板值；10级锁定精英怪生成概率为100%；15级使首领怪获得额外的护盾技能；20级以上赋予怪物极高的穿甲被动；最高级可能附带每回合自然掉血等极端惩罚。
- 作为补偿，在高阶噩梦中刷出的装备，其“物品强度（iPwr）”基础分提高，是产出顶级套装的首要途径。
