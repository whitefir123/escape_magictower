# 实体属性与角色职业设定字典

## 一、 实体基础属性词典 (Entities Stats Dictionary)
无论是玩家、小兵、精英还是首领，所有参与战斗的实体均由以下属性驱动。

### 1.1 主面板属性
- **生命值 (Health Points - HP) / 最大生命值 (Max HP)**：降至 0 则判定实体死亡。
- **法力值 (Mana Points - MP) / 最大法力值 (Max MP)**：释放大部分主动及终极技能的消耗资源。
- **物理攻击 (Physical ATK)**：决定白刃战及物理系技能的基础伤害源。
- **魔法攻击 (Magical ATK)**：决定法术技能、元素反应的基础伤害源。
- **物理防御 (Physical DEF)**：用于对抗并减免受到的物理伤害。
- **魔法防御 (Magical DEF)**：用于对抗并减免受到的魔法/元素伤害。

### 1.2 统一基础资源系统 (Unified Core Resources)
- **法力池 (Mana)**：用于释放常规的主动小技能与闪避。不同职业获取方式不同（如法师自然高回，剑客普攻回蓝，骑士受击回蓝）。
- **怒气槽 (Rage)**：全职业统一的次级战斗槽，仅用于释放必杀技（大招）。不可自动回复，必须依靠对敌人造成伤害或自身受到伤害积攒。

### 1.3 进阶战斗百分比 (Combat Modifiers)
- **暴击率 (Critical Rate)**：攻击造成爆伤的概率。
- **暴击伤害倍率 (Critical Multiplier)**：发生暴击时的伤害放大系数（基础通常为 1.4倍 ~ 1.5倍）。
- **防御穿透系数 (Armor Penetration / Magic Pen)**：攻击时，强制无视目标设定百分比的对应防御值。对抗高防御硬皮怪物时的核心攻坚属性。
- **闪避率 (Dodge)**：完全规避一次直接攻击的概率。（通常对持续性异常状态伤害无效）。

### 1.4 跨界属性协同方程 (Cross-Stat Synergy & Conversion Limits)
为了避免玩家在构筑时抽到包含“非本流派”主属性的装备/符文而产生强烈的废件感，系统底层植入了全实体通用的【属性跨界转化协定】。
为保证平衡与不喧宾夺主，所有转化曲线均遵循**边际效益极速递减（Diminishing Returns）**，且设有绝对硬上限（Hard Cap）：

1. **力量破法 (Force Breaches Magic)**：
   - **转化公式**：`额外魔法穿透率 = 物理攻击力 / (物理攻击力 + 500) * 20%`
   - **数值约束**：该机制提供的额外魔穿**绝对上限为 20%**。
2. **魔力附刃 (Arcane Edges)**：
   - **转化公式**：`额外物理穿透率 = 魔法攻击力 / (魔法攻击力 + 500) * 20%`
   - **数值约束**：该机制提供的额外物穿**绝对上限为 20%**。
3. **重甲霸体 (Armor Grants Tenacity)**：
   - **转化公式**：`额外异常控制抗性 = 物理防御力 / (物理防御力 + 1000) * 40%`
   - **数值约束**：该机制提供的额外状态抗性（减少被控时间）**绝对上限为 40%**。
4. **魔场感知 (Ward Grants Precision)**：
   - **转化公式**：`额外暴击率 = 魔法防御力 / (魔法防御力 + 800) * 15%`
   - **数值约束**：该机制提供的额外常驻暴击率**绝对上限为 15%**。

## 二、 角色池与职业特性 (Classes Configuration)
游戏首发提供4名风格明显的职业，拥有完全独立的底层基础架构与流派技能。每位英雄入局初始经验条固定为 **0/100**，并随等级平缓递增。

### 2.1 首发四大职业概览
1. **流浪剑客 (Vagabond Swordsman)**：均衡战士，依赖普攻回法力，拥有专属闪避[燕返]。
2. **奥术学徒 (Arcane Apprentice)**：法系核弹，自带极高的法力自动回复，拥有专属闪避[相位闪现]。
3. **暗影刺客 (Shadow Assassin)**：高移速高频暴击，背刺必定暴击破甲，拥有专属闪避[暗影步]。
4. **虔诚圣骑 (Devout Paladin)**：重装坦克，受击回蓝，自带临时白盾减伤机制，拥有专属闪避[举盾突骑]。
（各职业详细数值、双被动、双小技能及怒气大招见：`05_Hero_Classes_And_Skills.md`）

## 三、 怪物阵营与生态系 (Monsters Ecology)
除了拥有生命攻击等基础属性面板外，怪物的核心特征由其“种族标签”和“首领特性”决定。

### 3.1 怪物种族标签 (Monster Tags)
怪物天生携带着各类分类标签，如：**亡灵、构装、野兽、人形、元素、恶魔、自然、飞行** 等。
- 这些标签是玩家局外天赋和局内特定装备针对的靶点。比如：“圣银之剑”底材的装备天生自带对【亡灵】和【恶魔】的最终伤害 +50% 补正。

### 3.2 进阶定级机制
所有怪物分为：
- **普通怪**：标准属性成长底版模型。
- **精英怪**：任何普通小怪都有概率突变。体积强制放大1.5倍，强制追加[精英]标签，基础血量与双攻直接乘以 3~5 倍系数，并强制抽取 1~3 个隐藏受苦机制。突变使得经验值掉落倍增至 5.0 倍。
- **首领（Boss）**：关卡守门员。除了面板离谱，还具备独特的阶段技能树（例如半血狂暴），全状态异常抗性高，提供极高额的基础经验。
