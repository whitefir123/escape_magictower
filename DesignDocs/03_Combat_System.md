# 深度战斗与元素反应系统 (Deep Combat Mechanics)

## 一、 伤害判定与防拖沓结算逻辑 

### 1.1 类魔塔碰撞与索敌交互机制 (Collision & Targeting)
作为带有重度魔塔基因的地下城游戏，玩家与怪物的基础交锋规则被严格定义如下：
- **玩家的主动输出**：
  - **近战碰撞**：玩家向怪物所在的网格（或碰撞体）发起强制位移，系统即结算为玩家发起了一次"主动挥砍"。
  - **远程点选**：玩家通过右键单击远处的怪物发射投射物。一旦有怪物进入追击状态（仇恨连线），玩家角色会自动平A射击范围内最近的追击目标。
- **敌我反击的【不对称法则】**（Asymmetrical Counterattack）：
  - **怪物的两种反击形态**：
    1. **被动反击 (Passive Counter)**：无论怪物处于什么状态，只要玩家对其成功发起一次主动攻击（近战碰触或蓄意远程攻击），怪物会**瞬间无条件予以一次被动还击**（魔塔经典"换血"底层逻辑）。
    2. **主动追击 (Active Pursuit)**：玩家的主动攻击会激怒怪物，使其进入"追击状态"。追击中的怪物在它的行动回合（或攻击CD就绪时）会主动冲锋对玩家发起攻击。
  - **玩家的纯粹主动性**：**极其核心的防作弊/防挂机机制**！如果一只追击中的怪物主动跑来咬了玩家一口（发起了主动攻击），玩家**绝不会**像怪物那样也给它回敬一次"被动反击"。玩家只会单方面挨打并扣血。这奠定了玩家必须靠自己的"走位拉扯（Kite）"与"先手出击"去赢取伤害节奏的微操空间。

#### 1.1.1 格子移动与碰撞回弹精确定义 (Grid Movement & Bump Combat)

本项目为**像素类魔塔**，地图由离散正方形格子（Tile，1×1 Unity 单位）组成。所有实体（英雄、怪物）的位置**必须对齐到格子中心**。

**移动规则：**
- **格子步进**：玩家按 WASD（或方向键）时，尝试向该方向移动**一格**。移动表现为从当前格平滑插值到目标格（动画 0.1~0.2 秒）。
- **输入锁定**：移动动画播放期间不接受新的移动输入，防止跳格。
- **怪物也遵循格子移动**：怪物有独立的移动节拍（由 `attackInterval` 或专属 `moveInterval` 控制），追击状态下每隔 N 秒向玩家方向步进一格。

**移动前碰撞检测（碰撞回弹）：**
玩家尝试移动到目标格前，系统按以下优先级检测：
1. **目标格为墙壁或障碍物** → 移动取消，无事发生。
2. **目标格被怪物占据** → **碰撞回弹 + 战斗结算**：
   - **视觉表现**：玩家向目标格方向位移约半格（0.5 格），随后弹回原格。
   - **战斗结算**：玩家对该怪物造成一次普攻伤害 → 怪物**瞬间**被动反击一次 → 怪物进入追击状态。
3. **目标格为空地** → 移动成功，玩家占据新格子，原格释放。

**怪物碰撞玩家（不对称法则）：**
追击中的怪物尝试步进到玩家所在格时：
1. 移动失败（不可互相穿透），执行碰撞回弹动画。
2. 怪物对玩家造成一次主动攻击伤害。
3. **玩家绝不被动反击**——这是与玩家攻击的核心不对称。

### 1.2 五槽技能组与怒气锁定驱动 (5-Slot Skills & Rage)
旧设定中无脑全CD大招的松散技能树已被废弃。玩家职业构筑将强制采用极其紧凑的 5 槽标准：
- **被动槽位 (2个)**：常驻生效的底层机制光环（如：暴击率增加、或特定连击触发特写）。
- **主动小技能 (2个)**：由常规法力值（MP）或短CD驱动的战术动作（如火球、位移、陷阱）。
- **终极必杀技 (1个)**：拥有独立键位的破局大招（Ultimate）。
  - **大招释放绝对法则**：大招的冷却按钮被锁死，**完全不与时间流逝挂钩（无常规CD）**。只有当玩家通过普攻、挨打将左上角的 **"怒气条/连击点 (Rage/Combo)"** 积聚至 100% 满溢时，大招才允许被点击释放瞬间倾泻。这极大增强了终极技能的爆发战术含金量。（需与 `GameData_Blueprints` 内各职业的大招消耗表对齐实现）。

### 1.3 完整的伤害流转链路
一次有效的物理或魔法攻击从发起者到受击者，必须经历以下步骤结算：
1. **威力判定**：提取攻击侧面板及当前技能倍率（判断走物理池还是魔法池）。
2. **暴击检算**：掷骰子判断是否暴击。暴击不仅吃暴击伤害倍率乘区，还额外附带一次隐性的"无视部分防御（穿甲）"加权，确保暴击的价值在极高难度下不衰减。
3. **防御减免计算**：
   - **穿甲综合结算**：攻击方的总护甲穿透率 = `面板基础穿透率 + 跨界转换穿透率 (如法师靠力量白嫖的魔穿)`。
   - **有效防御力**：防守方的绝对有效防御 = `自身面板防御值 * (1 - 攻击方的总穿透率)`。
   - 随之将有效防御代入递减公式（保证防御堆再高也不可能达到100%免伤），折算出最终承伤比例。
4. **最终修正执行**：在此基数上附加上如"种族克制"、"元素反应倍率"、"装备额外附加固定真伤"等特殊补正，最后落地扣血。如果扣血后目标生命小于等于0，则触发死亡回调。

### 1.4 软狂暴：战斗疲劳机制 (Fatigue System)
为了解决回合制中常见的"坦克叠甲无限苟活磨死高难Boss"的无聊状况，强制规定了时长惩罚系统：
- **机制概述**：在单一场次战斗中，随回合数推移或现实时间叠加，怪物方将不断获得名为"狂热/疲劳"的层数。
- **具体影响**：每叠加一层，怪物全方面攻击力上升、且获得额外的防御穿透系数。
- **最终结果**：当层数叠加到一定程度（如10层），这在数值上等同于"秒杀机制"，逼迫玩家去研究进攻构筑而非单纯挨打。

## 二、 元素异常排列与链式连锁法则 (Elemental Reactions)
五种基础元素除了自身的独立作用外，两两相交更会衍生出复杂的连锁反应图谱。触发条件为：先给敌方附加底层状态（A），随后使用附带状态（B）的攻击命中目标即完成解构。

### 2.1 五大基础异常（A类引子）
1. **灼烧 (BURN 隶属火)**：最基础的DOT（持续伤害），按照目标最大生命值的特定百分比流血，持续数个回合。
2. **潮湿 (WET 隶属水)**：单独挂存时没有任何伤害和负面效果，它是引发最多种类强控或导电的前置水媒介。
3. **冰封 (FROZEN 隶属冰)**：顶级硬控。令目标处于绝对静止状态（无法攻击无法闪避），持续短暂的时间。
4. **感电 (SHOCK 隶属雷)**：单独存在时同样没有效果，它是雷暴天气的导体印记。
5. **中毒 (POISON 隶属毒)**：DOT伤害，不同于火系的不可叠加，毒素是可以无限叠加层数（Stack）的，层数越高，掉血越多。

### 2.2 复合反应法则（A+B结算规则）
1. **蒸发 (Vaporize)** = [灼烧(火)]遇上[水]。
   - **结果**：清空目标身上的灼烧，将此次水属性攻击的伤害瞬间翻倍。
2. **融化 (Melt)** = [灼烧(火)]遇上[冰]。
   - **结果**：同样的伤害放大反应，清空灼烧并打出超高乘区一击。
3. **超载 (Overload)** = [灼烧(火)]遇上[雷]。
   - **结果**：引发无视防御的真实爆炸。清空并以目标为中心产生极广的范围AOE炸击。
4. **冻结 (Freeze)** = [潮湿(水)]遇上[冰]。
   - **结果**：清空潮湿，无视对方的所有控制抗性，强行转入较长时间的[冰封]硬控。
5. **感电连营 (Electro-Charged)** = [潮湿(水)]遇上[雷]。
   - **结果**：这是一种持续存续的双混合状态。不消耗潮湿，每次目标受击或随回合数，天雷将顺着水劈向周围最近的多个目标，持续施加附带麻痹（低微停顿）的链式伤害。
6. **碎冰 (Shatter)** = [冰封(冰)]遇上[火]或重火器。
   - **结果**：目标提前解冻，但遭到暴力破冰惩罚，受到极其夸张的纯物理系巨额伤害扣减。
7. **毒爆术 (Venom Blast)** = [中毒(毒)]群遇上[火]。
   - **结果**：点燃毒气层。一次性清空目标身上累积的所有中毒层数，根据消耗的层数多少按比例换算为一次极为可怕的剧烈AOE绿火爆炸。
