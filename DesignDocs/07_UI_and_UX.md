# UI 交互交互与信息呈现规范 (UI & UX Flow)

## 一、 核心界面架构
游戏的所有信息流转依赖于清晰、不阻断核心操作的 UI 堆栈。

### 1.1 主菜单界面 (Main Menu)
- **视觉重心**：展示当前已解锁的最高层数、当前持有的水晶数量。
- **功用入口**：
  - **新游戏 (New Run)**：包含职业选择（战士、法师等）与难度测定（选择噩梦层级基数）。必须提供职业特性的简要文本说明。
  - **天赋树 (Talent Tree)**：展示树状节点。节点必须存在“未解锁”、“可解锁（水晶足够）”、“已解锁”三种视觉状态。
  - **每日挑战 (Daily Challenge)**：显示今天的固定词缀（Buff/Debuff）以及进入按钮。
  - **图鉴/成就 (Bestiary & Achievements)**：罗列已遭遇的怪物资料（包含抗性弱点）与成就解锁进度。

### 1.2 局内探险 HUD (In-Game HUD)
游戏画面的主体是地下城网格，边缘悬浮轻量级 HUD。
- **顶部信息栏**：当前楼层深度、当前金币组、当前的难度层级标识（如 Nightmare Level 5）。
- **底部状态盘**：
  - 角色生命槽（红）、法力/怒气槽（蓝/橙）、经验条（白）。
  - 各类技能图标（Q键、Space键），带有明显的冷却时间（CD）遮罩反馈与法力不足的置灰判据。
- **右侧/左侧事件流 (Combat Log / Mini-map)**：
  - 简易小地图：展示已探明的房间连线。
  - 战斗事件信息框。

### 1.3 关键系统交互面板 (System Panels)
当触及游戏核心养成时，必须弹出阻塞式的全屏或半屏 UI。

#### a) 背包与角色面板 (Inventory & Character Sheet)
- **纸娃娃系统 (Equipment Slots)**：严格设定为 6 大槽位：武器、头盔、护甲、手套、鞋子、饰品。玩家可以通过拖拽或点击进行装备替换。
- **属性结算清单**：动态实时显示经过“装备”、“符文”、“天赋”三层叠加后的最终核心属性（如：物理攻击 153、护甲穿透 22%）。
- **背包网格 (Grid)**：存放未装备的战利品、消耗品（血瓶）与强化材料。

#### b) 符文命运抉择 (Rune Draft)
- 玩家升级或触发强化符文时，强制暂停并弹出 3 张卡牌。
- 卡牌必须以极其鲜明的颜色边框区分“稀有度（凡品、稀有、罕见、史诗、传说、诅咒）”。
- 卡牌上具有明显的“标签（如 物理加成、机制触发、诅咒）”。
- 面板底部提供“消耗 XX 金币刷新”以及“放弃换取 XX 金币”的按钮。

#### c) 铁匠铺综合台 (Blacksmith Interface)
这是一个复合型面板，由顶部的标签页（Tabs）切换：
- **强化页 (Enhance)**：中心槽放入装备，显示成功率（如果是 100% 则显示必成），下方显示下一级的属性预览对比（绿字增量），以及需要的金币与材料数。
- **重铸页 (Reforge)**：放入装备，警示玩家该操作会改变基础品质但保留强化等级。
- **镶嵌页 (Socket)**：展示装备带有的孔数量，右侧展示背包中可用的宝石，两者可通过交互互填。

## 二、 交互反馈准则 (Feedback Mechanisms)
为了让游戏“有打击感”和“明确性”，以下反馈规则是强制的：
- **受击反馈**：物理受击必须有反馈；被硬控（如冰冻）模型必须覆盖冰晶贴图或调色。
- **飘字系统 (Floating Text)**：
  - 普通伤害为白色/红色小字。
  - 暴击伤害必须放大字号，且飘字左边有暴击图标，并带有震动感。
  - 元素反应（如“超载！”、“融化！”）必须有醒目的特殊艺术字体飘出。
- **拾取反馈**：掉落物在地上时，高品质装备的光柱颜色必须与其稀有度一致；拾取入包时要有清脆的金币入袋或金属碰撞 UI 提示。

## 三、 玩家输入与操作映射 (Player Input Mapping)
考虑到目前优先在 Windows 端实现并保留未来多端移植性，游戏采用经典的键鼠双栖操作模式。所有按键（除基础移动外）均允许玩家在设置面板自定义。

### 3.1 移动指令 (Movement)
- **主键盘移动**：默认使用 **W、A、S、D** 控制角色在场景中进行移动。
- **鼠标辅助移动**：支持**鼠标右键单击**地板空地，角色将自动移动至目标点。

### 3.2 交互与锁定指令 (Interaction & Targeting)
- **场景交互**：面对宝箱、门、神龛、铁匠等可交互实体时，默认按下 **F 键** 或使用 **鼠标左键单击** 触发互动UI。
- **战斗操作分化**：
  - **近战职业**：纯正的类魔塔机制，使用移动键（WASD）或右键直接**向怪物所在的格子发起“碰撞操作”**，即可触发并结算一次近战主动攻击。
  - **远程职业**：使用**鼠标右键单击**远处的敌方单位发起攻击。**解放双手的辅助设定**：一旦怪物被激怒进入追击状态，远程玩家将自动对最近的追击目标发起普攻（自动射击），无需玩家疯狂连续点击。
