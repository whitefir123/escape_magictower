# 山海装备与套装系统设定 (Shanhai Equipment & Set System)

在基础的前后缀随机池（白绿蓝紫黄红彩）之上，存在着完全独立、跳出常规品质体系的顶级造物——【山海装备】与【套装共鸣】。该类装备极度稀有，**山海装备没有品质（白绿蓝等）区分**，且拥有足以颠覆英雄底层机制的专属被动，设计上采用纯量化参数模型。

---

## 一、 山海武器库 (Shanhai Weapons)
山海武器通常会直接强化或者改变英雄某一个核心“小技能”或“被动”的形态与乘区公式。

### 1. 斩铁者·雷文 (Raven, The Iron Cleaver)
- **适配职业**：[流浪剑客] 专属大剑
- **固定基础面板面板**：物理攻击力 (ATK) `+ 120`，暴击率 (Crit Rate) `+ 10%`
- **山海特效 (Unique Passive)** - 【狂风之眼】：
  - 英雄在施放 **[小技能二：旋风斩]** 期间，移动速度不再受到 30% 的减速惩罚（变回 100% 正常移速）；
  - 旋风斩的AOE判定半径绝对值扩大 `2.0` 个身位；
  - 被旋风斩伤害判定的敌人，强制附加持续 `3.0` 秒的 **[流血]** 状态（每秒受到 `(0.2 * 英雄当前ATK)` 的真实物理伤害）。

### 2. 虚空裂隙法杖 (Staff of Void Rift)
- **适配职业**：[奥术学徒] 专属法杖
- **固定基础面板**：魔法攻击力 (MATK) `+ 150`，最大法力值 (Max Mana) `+ 100`
- **山海特效 (Unique Passive)** - 【微型奇点】：
  - **[小技能一：奥术飞弹]** 的发射数量从 3 枚增加至 `5` 枚，连弹发射间隔缩短 30%，且所有飞弹的追踪追踪检定半径扩大 100%；
  - 如果同一次施法中的 5 枚飞弹全部命中同一个敌方目标，该目标将被施加 `1.2` 秒的 **[眩晕]**（对同一目标内置绝对冷却 `5.0` 秒）。

### 3. 猩红之蛾双刃 (Crimson Moth Twinblades)
- **适配职业**：[暗影刺客] 专属双匕首
- **固定基础面板**：物理攻击力 (ATK) `+ 95`，暴击伤害倍率 (Crit Damage) `+ 60%`
- **山海特效 (Unique Passive)** - 【饮血归来】：
  - 每次成功触发刺客 **[被动一：死穴看破]**（即精准判定到背面发起的暴击）时，有 `35%` 的独立概率直接清零 **[小技能一：影袭]** 的当前剩余冷却时间。
  - 由于该特效存在，手法极高的玩家可以实现无限瞬移背刺的“永动机”循环。

### 4. 黎明守护者巨锤 (Dawnbreaker Greatmace)
- **适配职业**：[虔诚圣骑] 专属战锤
- **固定基础面板**：物理攻击力 (ATK) `+ 80`，魔法攻击力 (MATK) `+ 80`，物理防御 (DEF) `+ 40`
- **山海特效 (Unique Passive)** - 【满溢之光】：
  - **[小技能一：圣光惩击]** 命中所赋予的治疗效果翻倍（提升至 `(4% * Max HP)` 的真实回血）；
  - 当该次治疗溢出（即英雄处于满血状态）时，溢出的治疗量 `100%` 直接转化为该职业的专属资源 **【信仰护盾】白盾积蓄值**，从而让血条永垂不朽或者为大招积攒极高额爆破伤害。

## 二、 通用山海首饰库 (Universal Shanhai Accessories)
山海首饰不受职业限制，通常提供改变整个战场资源或者绝对保命机制的特效。

### 1. 贪婪王冠 (Crown of Greed)
- **装备部位**：头盔 / 项链 (随机生成其中之一部位)
- **固定基础面板**：最大生命值 (Max HP) `+ 250`，移动速度 `+ 5.0`
- **山海特效 (Unique Passive)** - 【资本支配】：
  - 英雄背包中每持有 `1000` 枚金币，英雄获得 `1%` 全伤害（包括物理/魔法/真实伤害）的最终倍率提升限制（最高提升上限为 `50%`）。
  - *负面代价 (Curse)*：英雄每次受到税前（减伤前）超过最大生命值 `10%` 的环境或怪物伤害时，将从背包中掉落当前总金币比例 `2%` 的金币散落在地（可重新拾取，但如果没捡起来就会永久丢失）。

### 2. 永恒沙漏 (Hourglass of Eternity)
- **装备部位**：戒指
- **固定基础面板**：冷却缩减 (CD Reduction) `+ 15%`，所有异常状态抗性 `+ 30%`
- **山海特效 (Unique Passive)** - 【时间逆行】：
  - 当英雄受到会导致生命值归零（致死判定）的伤害瞬间，免疫该次结算伤害并进入 **[时空凝滞]** 状态；
  - 凝滞期间（持续 `2.5` 秒），英雄变为金身无敌不可被选中状态，且完全无法行动/施法；
  - 凝滞结束瞬间，强行将自身生命值拉回到 `(30% * Max HP)` 的安全线。
  - *备注*：此被动拥有长达 `180.0` 秒的极长独立冷却时间（不吃装备CD缩减），冷却完毕前该戒指只是一件带有血量的白板。

## 三、 赐福共鸣系统 (Affix Resonance Set System)
本作不设传统的“绿装同名部位羁绊”，取而代之的是创新的【词缀共鸣机制】。
英雄拥有 6 个核心套装空位（分为：【武器】、【头盔】、【胸甲】、【手套】、【鞋靴】、【首饰】）。
每个流派套装对这 6 个部位都有特定的“词缀要求”。**每个部位给出了两种候选词缀，玩家只要该部位装备包含其中任意一个指定的词缀，即视为点亮了该部位的套装组件。**
**当点亮的部位数量达到 2件、4件、6件时**，将直接解锁强大的跨界流派大招。

### 1. 狂战士之怒 (Fury of the Berserker - 物理系极速吸血流)
- **触发条件 (点亮部位所需的指定词缀)**：
  - **[武器]**：【重刃的】 或 【...之吸血】
  - **[头盔]**：【壮容的】 或 【...之自愈】
  - **[胸甲]**：【坚韧的】 或 【...之反伤】
  - **[手套]**：【坚固的】 或 【...之减CD】
  - **[鞋靴]**：【闪避的】 或 【...之移速】
  - **[首饰]**：【必爆的】 或 【...之狂力】
- **两件套 (2件共鸣)**：除了所有的普攻吸血外，英雄的所有“消耗冷却时间或资源的物理小技能”在造成伤害时，也将获得等于 `(已损失生命值 5%)` 的直接回血效果。
- **四件套 (4件共鸣)**：在战斗中，生命值比例每低于满血状态 `10%`，英雄将获得 `(4% * 基础 ATK 面板)` 的物理攻击力动态加成，以及 `3%` 的攻击速度加成。
- **六件套 (6件共鸣)**：【血战到底】：
  - 当英雄当前生命值降底于 `30%` 的红血阙值时，获得永久的防打断等级极高的 **【绝对霸体】**（免疫一切冰冻、击退、眩晕控制）；
  - 触发霸体期间，所有的近战普通攻击在造成伤害后，会额外对目标斩下一刀等同于 `(5% * 英雄当前已损失生命值总额)` 的不可减免真实伤害。

### 2. 星体观测者 (Stargazer's Wisdom - 法系必爆冷却流)
- **触发条件 (点亮部位所需的指定词缀)**：
  - **[武器]**：【会心的】 或 【...之回蓝】
  - **[头盔]**：【抗魔的】 或 【...之减CD】

  - **[鞋靴]**：【抗魔的】 或 【...之自愈】
  - **[首饰]**：【扩蓝的】 或 【极速的】
- **两件套 (2件共鸣)**：每溢出（超过职业原初始上限倍数）的 `10` 点法力值，将转化为 `1.5%` 的全局魔法暴击伤害。
- **四件套 (4件共鸣)**：英雄每次成功施放消耗法力值的大技能后，在接下来的 `4.0` 秒内，所有法术技能必定触发暴击（魔法暴击率视为 100%）。此被动拥有 10.0 秒的内置冷却时间。
- **六件套 (6件共鸣)**：【真理之眼】：
  - 对处于任何硬控（包含 `[冰冻]`、`[眩晕]`、`[致盲]`、`[嘲讽]`）状态下的敌方单位，英雄造成的所有元素 / 魔法属性伤害将强制无视目标 `50%` 的魔法抗性 (MDEF)；
  - 若使用任意主动法力技能击杀了怪物（Last Hit），立即返还该次施法所消耗法力数值的 `80%`，并在原地触发一次同等耗蓝威力的零消耗奥术爆破。

### 3. 不毁之钢屏障 (Aegis of Indestructible Steel - 重装反伤流)
- **触发条件 (点亮部位所需的指定词缀)**：
  - **[武器]**：【破甲的】 或 【...之连击】

  - **[首饰]**：【极速的】 或 【...之护盾】
- **两件套 (2件共鸣)**：英雄开局必定获得一个厚度等于自身 `(最大生命值的 15%)` 的白盾，且只要英雄处于静止状态超过 3 秒，该护盾就会自动补满。
- **四件套 (4件共鸣)**：英雄的面板闪避率强制锁定为 `0%`，但每牺牲（装备上原本存在的） 1% 闪避率，就会转化为 `1.5%` 的全局物理与魔法复合最终伤害减免。
- **六件套 (6件共鸣)**：【移动铁处女】：
  - 全身装备叠加的“反伤”效果不再仅仅针对近战物理接触伤害，而是能够**反弹一切伤害源**（包括漫天飞舞的魔法弹幕、范围AOE地火、首领激光）；
  - 反弹的伤害值变更为 `(该次受到的税前总伤害 * 15% + 1.2 * 自身物理防御力)` 的绝对真实伤害。

### 4. 召唤使的仪典 (Ritual of the Summoner - 万灵长河统御套)

  - **[首饰]**：【必爆的】 或 【...之处决】
- **两件套 (2件共鸣)**：场上每存在一个属于你的【友方召唤物】（分身/幻影/宝宝），你自身的移动速度增加 `5%`，物理防御与魔法抗性提升 `10%`，上限 5 层。
- **四件套 (4件共鸣)**：你所有的召唤物将共享并同时继承你面板上的【暴击率】与【暴击伤害倍率】的 100% 增幅属性，且召唤物的攻击能够触发你身上的武器击中特效（如引发闪电链或冰缓效果）。
- **六件套 (6件共鸣)**：【灵魂殉道】：
  - 当英雄由于失误受到致命伤害（血量将要归零）时，如果场上有存活的召唤物，则立刻无情献祭掉最近的一个召唤物。英雄免疫此次死亡危机，并瞬间恢复等同于该召唤物 `100%` 血量的生命值。
  - 献祭触发后，场上所有剩余的其他召唤物进入长达 5.0 秒的【狂化】状态（全伤害、体型与攻击速度翻倍！）。

- **两件套 (2件共鸣)**：打破时间常理，英雄所有技能冷却时间的基础运算结算硬上限从标准的 40% 破格提升至 `65%`。
- **四件套 (4件共鸣)**：每次使用专属闪避动作（如燕返、暗影步等）并成功闪躲掉任何一次敌方攻击判定后，立即削减所有处于冷却中的【小技能】长达 `1.5` 秒的当前剩余冷区时间。
- **六件套 (6件共鸣)**：【无限火力镜像】：
  - 英雄释放任何【必杀技】（终极大招）之后，有 `50%` 的逆天概率使得这发大招 **【不需要进入冷却倒计时，并且直接全额返还刚才释放时扣除的法力/能量资源】**！
  - *负面代价*：为了换取这种时间特权，英雄所有的最大生命值上限（包括装备叠加）将被系统强制削减 `40%`，成为名副其实的只靠无极闪避和疯狂甩技能存活的玻璃刀尖舞者。
